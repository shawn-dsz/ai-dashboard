Glossary
Understanding your stats

Session

One interaction with Claude Code, start to close. Includes all messages and tool calls.

Message

Each exchange between you and Claude: your prompts and Claude's responses.

Tokens

The basic unit of text processing. Roughly 1 token ≈ 4 characters.

Input Tokens

Tokens in your messages, prompts, and files you share.

Output Tokens

Tokens in Claude's responses. Generally cost more than input tokens.

Cache Read

When Claude reuses previously processed context. Costs ~90% less than new tokens.

Cache Write

When new context is stored in cache. Slightly more expensive upfront but saves on reruns.

Cache Hit Rate

Percentage of cache tokens read vs. newly computed. Higher = more cost efficient.

Tool Calls

Actions Claude performs: reading files, running commands, searching the web, etc.

Streak

Consecutive days with at least one session. Shows consistency in your workflow.
